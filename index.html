¿Qué es una API REST?

Una API REST (Representational State Transfer) es un conjunto de reglas que permiten a sistemas interactuar con otros a través de HTTP. Es ampliamente usada en el desarrollo web por su sencillez, escalabilidad, y capacidad de independencia entre cliente y servidor. Las APIs REST operan sobre los métodos HTTP (GET, POST, PUT, DELETE) y son ideales para aplicaciones web distribuidas.
